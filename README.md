# 项目说明
> 本项目为将具有番茄检测能力的智能体与Fisco Bcos区块链进行结合的番茄溯源系统。

## 项目启动介绍
> 1. 项目中存在三个文件，分别是Solidity2Go、get-linker和src
> (vue项目中文件太多，只上传了主要代码，直接将文件替换新项目中的src目录，并下载依赖即可)
> 2. 项目启动了三个端口服务，分别是8080、8081和5173，确保三个端口未被占用
> 3. Solidity2Go文件实现了调用智能合约方法的功能。文件包含了溯源合约，部署后根据实际的角色私钥替换项目中account目录中对应的私钥文件，并修改配置文件中对应的配置即可;
>    get-linker文件实现了调用智能体api的功能，使用Coze平台作为智能体开发平台，用户可自行设计智能体，并修改配置文件中的智能体id和访问令牌，
